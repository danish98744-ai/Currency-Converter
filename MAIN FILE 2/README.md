# 💱 Currency Converter 

a simple currency converter  app made by DANISH KHAN using python and customtkinter

## what it does
- convert between 10 currencies in real time
- uses a free api to get live exchange rates
- shows conversion history (last 10)
- swap button to quickly reverse currencies
- dark mode user intreface

## currencies supported
- 🇮🇳 India (INR)
- 🇺🇸 United States (USD)
- 🇸🇦 Saudi Arabia (SAR)
- 🇦🇪 UAE (AED)
- 🇬🇧 United Kingdom (GBP)
- 🇪🇺 Europe (EUR)
- 🇯🇵 Japan (JPY)
- 🇨🇳 China (CNY)
- 🇨🇦 Canada (CAD)
- 🇦🇺 Australia (AUD)
